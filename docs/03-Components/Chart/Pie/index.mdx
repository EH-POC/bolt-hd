# Chart.Pie

Pie chart are used to show the proportions of categorical data, with the size of each piece representing the proportion of each category.

## Chart.Pie

### Basic

Basic usage example.

```jsx live
<Chart.Pie
  data={[
    { data: 1, label: 'Mon' },
    { data: 20, label: 'Tue' },
    { data: 30, label: 'Wed' },
    { data: 40, label: 'Thu' },
    { data: 50, label: 'Fri' },
  ]}
/>
```

### Sizes

There are three sizes of a `Chart.Pie`: small, medium (default) and large.

```jsx live
<Box sx={{ display: 'flex' }}>
  <Chart.Pie
    size="large"
    data={[
      { data: 10, label: 'Mon' },
      { data: 20, label: 'Tue' },
      { data: 30, label: 'Wed' },
      { data: 40, label: 'Thu' },
      { data: 50, label: 'Fri' },
    ]}
    sx={{ marginRight: 'large' }}
  />
  <Chart.Pie
    size="medium"
    data={[
      { data: 10, label: 'Mon' },
      { data: 20, label: 'Tue' },
      { data: 30, label: 'Wed' },
      { data: 40, label: 'Thu' },
      { data: 50, label: 'Fri' },
    ]}
    sx={{ marginRight: 'large' }}
  />
  <Chart.Pie
    size="small"
    data={[
      { data: 10, label: 'Mon' },
      { data: 20, label: 'Tue' },
      { data: 30, label: 'Wed' },
      { data: 40, label: 'Thu' },
      { data: 50, label: 'Fri' },
    ]}
  />
</Box>
```

### With title

Pie chart can be displayed with a title.

```jsx live
<Chart.Pie
```

Pie chart can be displayed with icon inside.

```jsx live

/>
```

### With custom central content

Pie Chart allow to custom the central content inside it.

```jsx live
function App() {
  const data = React.useMemo(
    () => [
      { data: 10, label: 'Mon' },
      { data: 20, label: 'Tue' },
      { data: 30, label: 'Wed' },
      { data: 40, label: 'Thu' },
      { data: 50, label: 'Fri' },
    ],
    []
  );
  const total = React.useMemo(
    () => data.reduce((total, item) => total + item.data, 0),
    [data]
  );
  const [selectedSlice, setSelectedSlice] = React.useState(null);
  return (
    <Chart.Pie
      data={data}
      itemConfig={{
        onMouseOver: (e, slice) => setSelectedSlice(slice),
        onMouseOut: () => setSelectedSlice(null),
      }}
      centralContentRenderer={() => {
        if (selectedSlice)
          return (
            <div>
              {selectedSlice.label}: {selectedSlice.data}
            </div>
          );
        return <div>Total: {total}</div>;
      }}
    />
  );
}
```

### With custom tooltip

Pie Chart allows customization of the tooltip content using the `tooltipRenderer` prop.

```jsx live
function App() {
  return (
    <Chart.Pie
      data={[
        { data: 10, label: 'Monday' },
        { data: 20, label: 'Tuesday' },
        { data: 30, label: 'Wednesday' },
        { data: 40, label: 'Thursday' },
        { data: 50, label: 'Friday' },
      ]}
      tooltipRenderer={(item) => (
        <Box sx={{ padding: 'xsmall' }}>
          <Typography.Title level={5} sx={{ marginBottom: 'xsmall' }}>
            {item.name}
          </Typography.Title>
          <Box sx={{ display: 'flex', alignItems: 'center' }}>
            <Box
              sx={{
                width: 12,
                height: 12,
                borderRadius: '50%',
                backgroundColor: item.color,
                marginRight: 'xsmall',
              }}
            />
            <Typography.Text>
              Value: <strong>{item.value}</strong>
            </Typography.Text>
          </Box>
          <Typography.Text sx={{ marginTop: 'xsmall' }}>
            Percentage: <strong>{item.percentage.toFixed(1)}%</strong>
          </Typography.Text>
        </Box>
      )}
    />
  );
}
```

### With segment label

#### Basic

Pie chart can be displayed with segment labels.

```jsx live
<Chart.Pie
  data={[
    { data: 15, label: 'Mon' },
    { data: 20, label: 'Tue' },
    { data: 25, label: 'Wed' },
    { data: 5, label: 'Thu' },
    { data: 30, label: 'Fri' },
    { data: 20, label: 'Sat' },
    { data: 10, label: 'Sun' },
  ]}
  segmentLabelConfig={{
    showSegmentLabel: true,
  }}
/>
```

#### With custom segment label

Pie chart can be displayed with custom segment label renderer.

```jsx live
<Chart.Pie
  data={[
    { data: 15, label: 'Mon' },
    { data: 20, label: 'Tue' },
    { data: 25, label: 'Wed' },
    { data: 5, label: 'Thu' },
    { data: 30, label: 'Fri' },
    { data: 20, label: 'Sat' },
    { data: 10, label: 'Sun' },
  ]}
  segmentLabelConfig={{
    showSegmentLabel: true,
    segmentLabelRenderer: (item) =>
      `${item.name}: ${item.value} (${item.percentage.toFixed(0)}%)`,
  }}
/>
```

#### Hide small segment label

By default, the segment label is not shown if the percentage is less than 5% and the segment count is greater than 8 to avoid label collision.

```jsx live
<Chart.Pie
  data={[
    { data: 15, label: 'Week 1' },
    { data: 1, label: 'Week 2' },
    { data: 25, label: 'Week 3' },
    { data: 5, label: 'Week 4' },
    { data: 30, label: 'Week 5' },
    { data: 2, label: 'Week 6' },
    { data: 10, label: 'Week 7' },
    { data: 12, label: 'Week 8' },
    { data: 15, label: 'Week 9' },
    { data: 11, label: 'Week 10' },
    { data: 10, label: 'Week 11' },
    { data: 10, label: 'Week 12' },
    { data: 10, label: 'Week 13' },
    { data: 10, label: 'Week 14' },
  ]}
  segmentLabelConfig={{
    showSegmentLabel: true,
  }}
/>
```

## Chart.MultiplePie

Chart.MultiplePie are a set of concentric circles which is used to visualize hierarchical relationships. Maximum number of charts is 4.

### Basic

```jsx live
<Chart.MultiplePie
  data={[
    { data: [10, 15, 5], label: 'Mon' },
    { data: [20, 20, 25], label: 'Tue' },
    { data: [25, 20, 25], label: 'Wed' },
    { data: [5, 15, 10], label: 'Thu' },
    { data: [30, 20, 30], label: 'Fri' },
  ]}
  labels={['Tom', 'Jerry', 'Spike']}
  data-test-id="multiple-pie-chart"
/>
```

### With title

Multiple Pie chart can be displayed with a title.

```jsx live
<Chart.MultiplePie
  data={[
    { data: [10, 15, 5], label: 'Mon' },
    { data: [20, 20, 25], label: 'Tue' },
    { data: [25, 20, 25], label: 'Wed' },
    { data: [5, 15, 10], label: 'Thu' },
    { data: [30, 20, 30], label: 'Fri' },
  ]}
  labels={['Tom', 'Jerry', 'Spike']}
  title="Multiple Pie Chart"
/>
```
