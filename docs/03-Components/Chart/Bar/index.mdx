# Chart.Bar

Reusable Bar chart components.

## Examples

### Basic

The basic chart just contain the chart with no legend, title and filter.

```jsx live
function BarChart() {
  const labels = ['May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Dec'];
  const data = [
    { label: '07th', data: [3, 4, 2, 2.5, 4, 3, 6] },
    { label: '14th', data: [3, 4, 2, 2.5, 5, 3, 3] },
    { label: '21st', data: [3, 4, 2, 2.5, 2, 3, 6] },
    { label: '25th', data: [3, 4, 2, 2.5, 6, 3, 4] },
    { label: '31st', data: [3, 4, 2, 2.5, 7, 3, 6] },
  ];

  return (
    <Box sx={{ height: 400 }}>
      <Chart.Bar
        data={data}
        xAxisConfig={{
          title: 'Horizontal Title',
        }}
        yAxisConfig={{
          title: 'Vertical Title',
          labels,
        }}
        data-test-id="bar-chart"
      />
    </Box>
  );
}
```

### With Navigation

The bar chart support navigation between the data. We have `navigationConfig.visible` props to make the navigation button appear.

and we have `navigationConfig.onNavigate` props to handle the navigation.

<Note title="Improvement UX" intent="info">
  In case, we want to improve the UX by not allow the user interact with
  navigation when they at the first page or last page. We can use{' '}
  <Highlight>navigationConfig.disabledButtons</Highlight> with{' '}
  <Highlight>prev</Highlight> or <Highlight>next</Highlight> value to disable
  that button.
</Note>

```jsx live
function BarChart() {
  const [currentIndex, setCurrentIndex] = React.useState(0);
  const numberOfColumns = 6;
  const data = [
    { label: '07th', data: [3, 4, 2, 2.5, 4, 3, 6] },
    { label: '14th', data: [3, 4, 2, 2.5, 5, 3, 3] },
    { label: '21st', data: [3, 4, 2, 2.5, 2, 3, 6] },
    { label: '25th', data: [3, 4, 2, 2.5, 6, 3, 4] },
    { label: '31st', data: [3, 4, 2, 2.5, 7, 3, 6] },
  ];

  const labels = ['May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Dec'];

  const navigationMapping = {
    0: 'prev',
    [`${labels.length - numberOfColumns}`]: 'next',
  };

  return (
    <Box sx={{ height: 400 }}>
      <Chart.Bar
        data={data.map((d) => ({
          ...d,
          data: d.data.slice(currentIndex, currentIndex + numberOfColumns),
        }))}
        xAxisConfig={{
          maxValue: 25,
          tick: {
            interval: 5,
          },
          step: 5,
        }}
        yAxisConfig={{
          labels: labels.slice(currentIndex, currentIndex + numberOfColumns),
        }}
        legendConfig={{
          selectable: true,
          viewMoreText: 'More',
          maxLegend: 4,
        }}
        navigationConfig={{
          visible: true,
          onNavigate: (direction) => {
            if (direction === 'next') {
              if (currentIndex + numberOfColumns >= labels.length) return;
              setCurrentIndex(currentIndex + 1);
            }
            if (direction === 'prev') {
              if (currentIndex === 0) return;
              setCurrentIndex(currentIndex - 1);
            }
          },
          disabledButtons: navigationMapping[currentIndex],
        }}
      />
    </Box>
  );
}
```

### With customized tooltip

The customizing Tooltip feature allows users to tailor the appearance.

```jsx live
function BarChart() {
  const labels = ['May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Dec'];
  const data = [
    { label: '07th', data: [3, 4, 2, 2.5, 4, 3, 6] },
    { label: '14th', data: [3, 4, 2, 2.5, 5, 3, 3] },
    { label: '21st', data: [3, 4, 2, 2.5, 2, 3, 6] },
    { label: '25th', data: [3, 4, 2, 2.5, 6, 3, 4] },
    { label: '31st', data: [3, 4, 2, 2.5, 7, 3, 6] },
  ];

  return (
    <Box sx={{ height: 400 }}>
      <Chart.Bar
        data={data}
        xAxisConfig={{
          title: 'Horizontal Title',
        }}
        yAxisConfig={{
          title: 'Vertical Title',
          labels,
        }}
        data-test-id="customized-tooltip-bar-chart"
        tooltipRenderer={({ label, legendColor, value, yAxisLabel }) => {
          const serie = data.find((item) => item.label === label);

          const sum =
            serie !== undefined
              ? serie.data.reduce((sum, currentValue) => sum + currentValue, 0)
              : 0;

          const percent = ((value / sum) * 100).toFixed(2);
          const percentString = `(${percent}%)`;
          return (
            <>
              <Box
                data-test-id="legend-circle"
                style={{
                  backgroundColor: legendColor,
                  width: theme.sizes.xsmall,
                  height: theme.sizes.xsmall,
                  borderRadius: theme.radii.medium,
                }}
              />
              <Typography.Text
                fontSize={14}
                fontWeight="light"
                intent="subdued"
                tagName="span"
              >
                {label}
              </Typography.Text>
              <Typography.Text
                fontSize={14}
                fontWeight="light"
                intent="subdued"
                tagName="span"
              >
                {yAxisLabel}
              </Typography.Text>
              <Typography.Text fontSize={14} fontWeight="light" tagName="span">
                {value}
              </Typography.Text>
              <Typography.Text fontSize={14} fontWeight="light" tagName="span">
                {percentString}
              </Typography.Text>
            </>
          );
        }}
      />
    </Box>
  );
}
```

### With bar sizing

The bar chart can change the bar height by barConfig.size. We have two values is `medium` or `large`.
The default value is `large`.

