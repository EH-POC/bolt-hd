# Timeline

Timeline is used to show a series of information which is connected or ordered by time.

## Examples

### Basic

Providing a list of items.

```jsx live
<>
  <Timeline
    items={[
      {
        id: 1,
        text: 'Self Implementation',
        helpText:
          'Continue with your full access to all feature implementation.',
      },
      {
        id: 2,
        text: '24 January',
        helpText:
          "You trial will end. You will revert back to your plan's premium features.",
      },
    ]}
  />
</>
```

### With intents

Providing a list of items with intent prop.

```jsx live
<>
  <Timeline
    items={[
      {
        id: 1,
        text: 'Default',
        helpText: 'Default description',
      },
      {
        id: 2,
        text: 'Success',
        helpText: 'Success description',
        intent: 'success',
      },
      {
        id: 3,
        text: 'Info',
        helpText: 'Info description',
        intent: 'info',
      },
      {
        id: 4,
        text: 'Warning',
        helpText: 'Warning description',
        intent: 'warning',
      },
      {
        id: 5,
        text: 'Danger',
        helpText: 'Danger description',
        intent: 'danger',
      },
      {
        id: 6,
        text: 'Error',
        helpText: 'Error description',
        intent: 'error',
      },
    ]}
  />
</>
```
