# DatePicker

A DatePicker shows a monthly calendar and allows the user to choose a single date.

## Examples

### Basic

Simple usage of DatePicker.

```jsx live
function BasicDatePicker() {
  const [value, setValue] = React.useState('19/02/2021');
  return (
    <DatePicker
      value={value}
      onChange={(newValue) => setValue(newValue)}
      id="basic-date-picker"
    />
  );
}
```

### With limited range.

Dates can be limited in a range between `minDate` and `maxDate`.

```jsx live
function LimitedDatePicker() {
  const [value, setValue] = React.useState('19/02/2021');
  return (
    <DatePicker
      value={value}
      onChange={(newValue) => setValue(newValue)}
      minDate={new Date(2021, 1, 3)}
      maxDate={new Date(2021, 1, 20)}
      id="limited-range-date-picker"
    />
  );
}
```

### DatePicker.Range

Be used to select a date range instead of a single date.

```jsx live
function DatePickerRange() {
  const [value, setValue] = React.useState({
    startDate: '03/02/2021',
    endDate: '03/03/2021',
  });
  const ids = {
    startDate: 'date-range-start',
    endDate: 'date-range-end',
  };
  return (
    <DatePicker.Range
      value={value}
      onChange={({ startDate, endDate }) => setValue({ startDate, endDate })}
      id={ids}
    />
  );
}
```

### DatePicker.Week

Be used to select a week instead of a single date.

#### Basic

```jsx live
function WeekPicker() {
  const [value, setValue] = React.useState('Week 01, 2021');
  return (
    <DatePicker.Week
      value={value}
      onChange={(newValue) => setValue(newValue)}
      id="week-picker"
    />
  );
}
```

#### Custom week

`DatePicker.Week` is allow to custom the start day of week via `weekStartsOn`.

```jsx live
function WeekPicker() {
  const [value, setValue] = React.useState('Week 01, 2021');
  return (
    <DatePicker.Week
      value={value}
      onChange={(newValue) => setValue(newValue)}
      id="week-picker-custom-week"
      weekStartsOn={5}
    />
  );
}
```

### DatePicker.Fortnightly

Be used to select a fortnightly instead of a single date.

#### Basic

```jsx live
function FortnightlyPicker() {
  const [value, setValue] = React.useState('03/01/2021');
  return (
    <>
      <DatePicker.Fortnightly
        value={value}
        onChange={setValue}
        id="fortnightly-picker"
        placeholder="Fortnightly"
        minDate={new Date(2021, 0, 1)}
        maxDate={new Date(2021, 5, 31)}
      />
    </>
  );
}
```

#### Custom week

`DatePicker.Fortnightly` is allow to custom the start day of week via `weekStartsOn`.

```jsx live
function FortnightlyPicker() {
  const [value, setValue] = React.useState('04/01/2021');
  return (
    <DatePicker.Fortnightly
      value={value}
      onChange={setValue}
      id="fortnightly-picker"
      placeholder="Fortnightly"
      minDate={new Date(2021, 0, 1)}
      maxDate={new Date(2021, 5, 31)}
      weekStartsOn={1}
    />
  );
}
```

### DatePicker.SingleMonth


### DatePicker.FixedRange

`DatePicker.FixedRange` is used to custom select start day and auto select end day with range option.

```jsx live
function FixedRangeExample() {
  const [value, setValue] = React.useState('04/02/2021');
  return (
    <DatePicker.FixedRange
      value={value}
      onChange={setValue}
      range="3w"
      id="fixed-range-picker"
    />
  );
}
```
