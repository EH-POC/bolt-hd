# Tabs

Tabs make it easy to switch between different views.

## When to Use

Can be used to manage contents when you have many different categories on the same page.

## Examples

### Basic

```jsx live
function BasicTabs() {
  const tabs = [
    { id: 1, title: 'Tab 1', panel: 'Tab content 1' },
    { id: 2, title: 'Tab 2', panel: 'Tab content 2' },
    { id: 3, title: 'Tab 3', panel: 'Tab content 3', disabled: true },
    { id: 4, title: 'Tab 4', panel: 'Tab content 4' },
  ];
  const [selectedTabId, setSelectedTabId] = React.useState(1);
  return (
    <Tabs
      id="sample-tabs"
      tabs={tabs}
      selectedTabId={selectedTabId}
      onChange={(newTabId) => setSelectedTabId(newTabId)}
    />
  );
}
```

### With secondary Tabs

Use `Tabs.Sub` for nested tabs (secondary).

```jsx live
function SecondaryTabs() {
  const [selectedSubTabId, setSelectedSubTabId] = React.useState(4);
  const [selectedTabId, setSelectedTabId] = React.useState(1);
  const subTabs = [
    { id: 1, title: 'Sub Tab 1', panel: 'SubTab content 1' },
    { id: 2, title: 'Sub Tab 2', panel: 'SubTab content 2' },
    { id: 3, title: 'Sub Tab 3', panel: 'SubTab content 3', disabled: true },
    { id: 4, title: 'Sub Tab 4', panel: 'SubTab content 4' },
  ];
  const panelWithSubTab = (
    <Tabs.Sub
      id="smaple-sub-tabs"
      tabs={subTabs}
      selectedTabId={selectedSubTabId}
      onChange={(newTabId) => setSelectedSubTabId(newTabId)}
    />
  );
  const tabs = [
    { id: 1, title: 'Tab 1', panel: panelWithSubTab },
    { id: 2, title: 'Tab 2', panel: 'Tab content 2' },
    { id: 3, title: 'Tab 3', panel: 'Tab content 3', disabled: true },
    { id: 4, title: 'Tab 4', panel: 'Tab content 4' },
  ];
  return (
    <Tabs
      id="sample-tabs"
      tabs={tabs}
      selectedTabId={selectedTabId}
      onChange={(newTabId) => setSelectedTabId(newTabId)}
    />
  );
}
```
