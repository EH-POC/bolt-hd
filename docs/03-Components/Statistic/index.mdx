# Statistic

Display statistic number.

## Examples

### Text alignment justified

- There are 2 `align` options : `left` & `right`.
- `right` will be set by default.

```jsx live
function JustifiedStatistic() {
  return (
    <StatisticStyledContainer>
      <div style={{ margin: 0, padding: 0, width: '100%' }}>
        <Statistic value="$8000.00" title="Hero Dollar Balance" align="left" />
        <Button text="Top up" />
      </div>
      <Statistic value="$80.00" title="Hero Dollar Balance" />
    </StatisticStyledContainer>
  );
}
```

### Units

- We can add `prefix` or `suffix`.
- **prefix** which is **unit before the value**.
- **suffix** which is **unit after the value**.

```jsx live
function UnitStatistic() {
  return (
    <StatisticStyledContainer>
      <Statistic
        value="12,345"
        title="Active Users"
        align="left"
        prefix={<Icon icon="raising-hands" intent="primary" />}
      />
      <Statistic value="15" title="Sync Errors" suffix="/ 1000" align="left" />
      <Statistic
        value="0"
        title="Submitted documents"
        prefix={<Icon icon="document-outlined" intent="success" />}
        suffix="/ 20"
      />
    </StatisticStyledContainer>
  );
}
```

### With card examples

- `text` **intent** will be set as default for **light card background color**.
- `white` **intent** may be used for **dark card background color**.
- `inherit` **intent** may be used for **custom color**.
