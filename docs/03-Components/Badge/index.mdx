# Badge

- Badge is used to describe status or value of a text or number.
- It can be used as standalone status; or a notification.

## Examples

There are 3 types: **Standalone**, **Count** and **Status**.

### Standalone

#### Size

There are 2 sizes: **Medium** and **Small**. The default size is **Medium**.

```jsx live
<>
  <Box
    sx={{
      mb: 'medium',
      display: 'flex',
      gap: 'medium',
      flexWrap: 'wrap',
      alignItems: 'flex-end',
    }}
  >
    <Badge intent="primary" content="PRIMARY" size="small" />
    <Badge intent="primary" content={5} size="small" />
    <Badge intent="primary" content={45} size="small" />
  </Box>
  <Box
    sx={{
      mb: 'medium',
      display: 'flex',
      gap: 'medium',
      flexWrap: 'wrap',
      alignItems: 'flex-end',
    }}
  >
    <Badge intent="primary" content="PRIMARY" size="medium" />
    <Badge intent="primary" content={5} size="medium" />
    <Badge intent="primary" content={45} size="medium" />
  </Box>
</>
```

### Count

It can be **Number** or **Icon**.

#### Icon

```jsx live
<Badge.Count content="clock-outlined">
  <Box
    backgroundColor="grey-light-60"
    style={{
      width: theme.sizes.xxxlarge,
      height: theme.sizes.xxxlarge,
      borderRadius: '50%',
      background: theme.colors.palette.greyLight60,
      display: 'inline-block',
      verticalAlign: 'middle',
    }}
  />
</Badge.Count>
```

#### Number

`99+` is displayed when count is larger than 99.
