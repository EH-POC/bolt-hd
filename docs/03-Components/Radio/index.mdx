# Radio

- The radio usually prepresents an option in a list where only one item can be selected at a time.

- Hero design supports 2 styles of radio: Radio & Radio.Button. In addition, we support Radio.Group & Radio.ButtonGroup to prepresent each radio style in groups.

## Examples

### Basic

Basic usage of a radio.

```jsx live
<>
  <Radio name="basic_usage" value="hr_document" text="HR Document" />
  <Radio name="basic_usage" value="hr_guide" text="HR Guide" />
</>
```

### Disabled

Show a disabled option.

```jsx live
<>
  <Radio
    value="independent_contractor"
    text="Independent contractor"
    disabled
    checked
  />
  <Radio value="permanent_employee" text="Permanent employee" disabled />
</>
```

### Controlled checked status

```jsx live
function RadioStatus() {
  const [checked, setChecked] = useState(false);
  return (
    <Radio
      value="hr_guide"
      text="HR Guide"
      checked={checked}
      onChange={(e) => setChecked(e.target.checked)}
    />
  );
}
```

### Radio.Group

Generate a group of radios from an array to show a series of options.

```jsx live
function RadioGroup() {
  const [value, setValue] = useState('hr_document');
  return (
    <Radio.Group
      value={value}
      name="radio_group"
      onChange={setValue}
      options={[
        { text: 'HR Document', value: 'hr_document' },
        { text: 'HR Guide', value: 'hr_guide' },
        {
          text: 'Indenpendent Contractor',
          value: 'independent_contractor',
          disabled: true,
        },
      ]}
    />
  );
}
```

### Radio.Button

Radio can be rendered as a button with Radio.Button.

```jsx live
function RadioButton() {
  const [value, setValue] = useState('policy_document');
  return (
    <>
      <Radio.Button
        name="radio_button"
        value="hr_document"
        text="HR Document"
        style={{ marginRight: theme.space.small }}
        checked={value === 'hr_document'}
        onChange={(e) => {
          if (e.target.checked) {
            setValue('hr_document');
          }
        }}
      />
      <Radio.Button
        name="radio_button"
        value="hr_guide"
        text="HR Guide"
        style={{ marginRight: theme.space.small }}
        checked={value === 'hr_guide'}
        onChange={(e) => {
          if (e.target.checked) {
            setValue('hr_guide');
          }
        }}
      />
      <Radio.Button
        name="radio_button"
        value="independent_contractor"
        text="Independent Contractor"
        disabled
        style={{ marginRight: theme.space.small }}
        checked={value === 'independent_contractor'}
        onChange={(e) => {
          if (e.target.checked) {
            setValue('independent_contractor');
          }
        }}
      />
      <Radio.Button
        name="radio_button"
        value="policy_document"
        text="Policy Document"
        disabled
        checked={value === 'policy_document'}
        onChange={(e) => {
          if (e.target.checked) {
            setValue('policy_document');
          }
        }}
      />
    </>
  );
}
```
