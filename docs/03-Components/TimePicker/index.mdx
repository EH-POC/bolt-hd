# TimePicker

A TimePicker let an user choose a time value from a respective dropdown to select a time value, with support for showing seconds and toggling between 12-hour and 24-hour systems.

## Examples

### Basic

Simple usage of TimePicker.

```jsx live
function BasicTimePicker() {
  const [value, setValue] = React.useState();
  return (
    <TimePicker
      value={value}
      onChange={(newValue) => setValue(newValue)}
      id="time-picker__basic"
    />
  );
}
```

### With seconds

TimePicker with seconds.

```jsx live
function TimePickerWithSecond() {
  const [value, setValue] = React.useState();
  return (
    <TimePicker
      withSeconds
      value={value}
      onChange={(newValue) => setValue(newValue)}
      id="time-picker__with-seconds"
    />
  );
}
```

```jsx live

}
```
