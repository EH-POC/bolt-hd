# Divider

A divider is used to separate contents or UI sections.

## Examples

Both horizontal and vertical margin of Divider can be set with 5 options as below:

| Option | Margin |
| ------ | ------ |
| xsmall | 4      |
| small  | 8      |
| medium | 16     |
| large  | 24     |
| xlarge | 32     |

As default, Divider's `left/right` and `top/bottom` margin would be `0px`.

### Basic

Divider can be used to separate a header and a paragraph.

```jsx live
<>
  <Typography.Title level={2}>Employment Hero</Typography.Title>
  <Divider marginY="medium" />
  <Typography.Text>
    Employment Hero is Australia&apos;s first all-in-one HR software, Payroll
    and benefits platform for employers & employees.
  </Typography.Text>
</>
```

### Line pattern

Divider supports 2 line style `solid` (default) and `dashed` via `pattern` prop.

```jsx live
<>
  <Typography.Title level={2}>Employment Hero</Typography.Title>
  <Divider marginY="medium" pattern="solid" />
  <Typography.Text>
    Employment Hero is Australia&apos;s first all-in-one HR software, Payroll
    and benefits platform for employers & employees.
  </Typography.Text>
  <Divider marginY="small" pattern="dashed" />
</>
```
