# Filters

Filters is a composite component that filters the items of a list or table.

## Features

- [Default settings](#default-settings-using-with-table) using with Table.
- [Customized filter apply changes](#customized-filter-apply-changes).
- [Customized filter apply all changes](#customized-filter-apply-all-changes).

## Examples

### Default settings using with Table

:::tip

For typescript usage, ref to [FiltersTableExample.tsx](https://github.com/Thinkei/hero-design/blob/master/packages/website/src/components/Filters/FiltersTableExample.tsx)

:::

        accessor: 'status',

          const intent = value === 'Approved' ? 'success' : 'warning';
          return (
            <Tag
              text={value.toUpperCase()}
              variant="outlined"

            />

        },

### Filter.Group with wide layout

When number of filters inside a Filter.Group exceed 5, a wide layout will be used to render filters. This layout is available only on large and xlarge screens.
