# Box

Box is an abstract component that allows to:

- Create layout for its content.
- Access theme easily via props instead of inline style or css in js methods.

## Examples

### Basic

Box can be use as a layout wrapper with margin and padding.

```jsx live
<Box m="medium" p="medium">
  <Typography.Text>This is box content</Typography.Text>
</Box>
```

### With colors

Box offers colors for background, border and text content.

```jsx live
<Box p="medium" bgColor="green-light-90" color="green" fontSize="medium">
  This is box content
</Box>
```

### With borders

Box offers shortcuts for border via `borderWidth`, `borderStyle`, `borderRadius` and `borderColor`.

```jsx live
<Box
  p="medium"
  borderWidth="base"
  borderStyle="solid"
  borderColor="grey-light-60"
  borderRadius="base"
  fontSize="medium"
>
  <Typography.Text>This is box content</Typography.Text>
</Box>
```

### With typography

Box offers shortcuts for text via `fontSize`, `fontWeight`, `lineHeight` and `color`.

```jsx live
<Box
  p="medium"
  borderWidth="base"
  borderStyle="solid"
  borderColor="grey-light-60"
  borderRadius="base"
  fontSize="medium"
  lineHeight="medium"
  fontWeight="semi-bold"
  color="grey-dark-75"
>
  This is box content
</Box>
```

### With layout

Box supports different display values to control layout behavior.

```jsx live
<Box p="medium" bgColor="grey-light-90">
  <Typography.Text>Block display (default):</Typography.Text>
  <Box display="block" p="small" m="small" bgColor="green-light-90">
    This is a block box
  </Box>
  <Box display="block" p="small" m="small" bgColor="green-light-90">
    This is another block box
  </Box>

  <Typography.Text mt="medium">Inline display:</Typography.Text>
  <Box display="inline" p="small" m="small" bgColor="green-light-90">
    This is an inline box
  </Box>
  <Box display="inline" p="small" m="small" bgColor="green-light-90">
    This is another inline box
  </Box>
</Box>
```

### With flexbox

Box supports all flexbox properties for creating flexible layouts.

```jsx live
<Box
  display="flex"
  flexDirection="row"
  justifyContent="space-between"
  alignItems="center"
>
  <Box p="small" bgColor="green-light-90">
    Item 1
  </Box>
  <Box p="small" bgColor="green-light-90">
    Item 2
  </Box>
  <Box p="small" bgColor="green-light-90">
    Item 3
  </Box>
</Box>
```

### With position

Box supports positioning properties for creating complex layouts.

```jsx live
<Box
  position="relative"
  bgColor="grey-light-90"
  sx={{ height: 100, width: 500 }}
>
  <Box
    position="absolute"
    top="medium"
    right="medium"
    p="medium"
    bgColor="green-light-90"
  >
    Absolute positioned box
  </Box>
  <Box
    position="absolute"
    bottom="medium"
    left="medium"
    p="medium"
    bgColor="red-light-90"
  >
    Another absolute positioned box
  </Box>
</Box>
```

### Render Box as HTML Tag

You can render Box Components as HTML Tag like `span`, `button`,...Default tag is `div`

```jsx live
<Box p="medium" tagName="code">
  This should be displayed as code
</Box>
```
