# Basic Table

- A table displays a collection of structured data in rows.
- Help users get an at-a-glance summary of the data to perform actions or navigate to a full-page representation of it.
- Organise information in an easy to scan way so that users can look for patterns and develop insights from data.

<Note title="Typescript usage" intent="error">
  Currently, our the Table component can support any type of props. So please
  use the typescript inside <Highlight>@ehrocks/hero-design-react</Highlight>.
  If you want to have any type of props, please create a new issue to let us
  know.
</Note>

## Examples

### Basic

#### Render the most basic table

By provide `data` & `columns`.

<Note title="Notes" style={{ marginTop: theme.space.large }}>
  <Highlight>accessor</Highlight> is optional, but when it is defined the column
  default sorting will be enabled by default.
</Note>

```jsx live
function BasicTabs() {
  function BasicTable({ size }) {
    const columns = React.useMemo(
      () => [
        {
          Header: 'Name',
          accessor: 'name',
          Cell: ({ value }) => (
            <Button.Link href="#" text={value} target="_blank" />
          ),
        },
        { Header: 'Age', accessor: 'age' },
        { Header: 'Status', accessor: 'status', disableSortBy: true },
      ],
      []
    );
    const data = [
      { name: 'Vy Nguyen', age: 50, status: 'Single' },
      { name: 'Hau Dao', age: 51, status: 'Single' },
      { name: 'Hanh Le', age: 80, status: 'Complicated' },
      { name: 'Tuan Thieu', age: 99, status: 'In relationship' },
      { name: 'Vinh Tony', age: 21, status: 'Single' },
    ];
    return <Table data={data} columns={columns} selectable size={size} />;
  }

  const tabs = [
    { id: 1, title: 'Default Size Table', panel: <BasicTable /> },
    {
      id: 2,
      title: 'Compact Size Table',
      panel: <BasicTable size="compact" />,
    },
  ];
  const [selectedTabId, setSelectedTabId] = React.useState(1);
  return (
    <Tabs
      id="basic-table-tabs"
      tabs={tabs}
      selectedTabId={selectedTabId}
      onChange={(newTabId) => setSelectedTabId(newTabId)}
    />
  );
}
```

### Accessing the original row data in a cell.

It is common to access to other fields of row data to handle logic in a customised cell. The row data object from the `data` array is also passed as the prop `row.original` of `Cell`.

```jsx live
function TableRowData() {
```

### Pagination

Pagination is one of table advanced feature. To enable it, we need to:

- Passing `pagination` configuration.
- Provide `onPaginationChange` callback to control the selected page value.

<Note title="Pagination index">
  Pagination index is 1-based which means the first page will be 1 (NOT 0).
</Note>

We may display items per page selection by:

- Passing `itemsPerPage` configuration.
- Provide `onItemsPerPageChange` callback to control the selected items per page.

<Note title="Items per page is only visible when:">
  Items per page UI is only visible when <Highlight>pagination</Highlight>{' '}
  configuration is defined.
</Note>


### Rows grouped by condition

Rows can be grouped by a condition by passing `groupBy` function to `groupingConfig` prop.

- [Optional] Specify `collapsedByDefaultGroups` to collapse the group by default.

<Note title="Renderer using" intent="warning">
  If you use custom render group header. The collapsed button will added by
  default at first column.
</Note>

```jsx live
function RowsGroupedBy() {
  const [selectedRows, setSelectedRows] = React.useState({
    0: true,
    2: true,
  });
  const data = [
    { name: 'Vy Nguyen', age: 50, status: 'Single' },
    { name: 'Hau Dao', age: 51, status: 'Single' },
    { name: 'Hanh Le', age: 80, status: 'Complicated' },
    { name: 'Tuan Thieu', age: 99, status: 'In relationship' },
    { name: 'Vinh Tony', age: 21, status: 'Single' },
    { name: 'Danh Pham', age: 45, status: 'In relationship' },
  ];
  const columns = React.useMemo(
    () => [
      {
        Header: 'Name',
        accessor: 'name',
        Cell: ({ value }) => (
          <Button.Link href="#" text={value} target="_blank" />
        ),
        Footer: 'Total',
      },
      { Header: 'Age', accessor: 'age' },
      {
        Header: 'Status',
        accessor: 'status',
        disableSortBy: true,
        Footer: () => `${data.length} users`,
      },
    ],
    []
  );
  return (
    <Table
      data={data}
      columns={columns}
      selectable
      selectedRows={selectedRows}
      onSelectedRowsChange={setSelectedRows}
      bulkCheckSettings={{
        disableCheckRule: (row) =>
          row.original.status === 'Complicated' || row.original.age === 45,
      }}
      groupingConfig={{
        groupBy: (data) => ({
          'Single and older than 50': data.filter(
            (row) => row.original.status === 'Single' && row.original.age > 50
          ),
          'Single and younger than 51': data.filter(
            (row) => row.original.status === 'Single' && row.original.age <= 50
          ),
          Complicated: data.filter(
            (row) => row.original.status === 'Complicated'
          ),
          'In relationship': data.filter(
            (row) => row.original.status === 'In relationship'
          ),
        }),
        collapsedByDefaultGroups: ['Complicated'],
        groupRenderer: {
          'Single and younger than 51': ({ groupedRows }) => {
            const names = groupedRows
              .map((row) => row.original.name)
              .join(', ');
            const ages = groupedRows.map((row) => row.original.age).sort();
            const firstAge = ages[0];
            const lastAge = ages[ages.length - 1];
            return (
              <>
                <td
                  colSpan={1}
                  style={{ padding: `${theme.space.large}px 0`, margin: 0 }}
                >
                  <Typography.Text>{`Name: ${names}`}</Typography.Text>
                </td>
                <td
                  colSpan={2}
                  style={{ padding: `${theme.space.large}px 0`, margin: 0 }}
                >
                  <Typography.Text>{`Age range: ${firstAge} - ${lastAge}`}</Typography.Text>
                </td>
              </>
            );
          },
        },
      }}
    />
  );
}
```
