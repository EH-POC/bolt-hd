# Carousel

- The carousel component is automatically scaled with the container that user set.
- Commonly used for slide of pictures.

## Examples

### Example format of Carousel collection

```js
const collection = [
  { id: 1, content: <img src="url_1" alt="imgage 1" /> },
  { id: 2, content: <img src="url_2" alt="imgage 2" /> },
  { id: 3, content: <img src="url_3" alt="imgage 3" /> },
];
```

### Basic usage of Carousel.

It has 2 variants `light` vs `dark` (to indicate dots button color)

- ** light ** (by default) is used for dark background color.
- ** dark** is used for light background color.

<Note title="Notes">
  <Highlight>selectedItemIndex, onChange</Highlight> are optional, but when they
  are defined the component turns into controlled.
</Note>

### Autoplay Carousel

- Can set interval in milliseconds, default interval is `2000`.
- On mouse hover, the animation will pause.

```jsx live
function AutoplayCarousel() {
  const { Title } = Typography;
  const [currentItemIndexDark, setCurrentItemIndexDark] = React.useState(0);
  const [currentItemIndexLight, setCurrentItemIndexLight] = React.useState(0);
  const darkBgCollection = [
    {
      id: 1,
      content: (
        <DarkCarouselWrapper>
          <Title intent="subdued">1</Title>
        </DarkCarouselWrapper>
      ),
    },
    {
      id: 2,
      content: (
        <DarkCarouselWrapper>
          <Title intent="subdued">2</Title>
        </DarkCarouselWrapper>
      ),
    },
    {
      id: 3,
      content: (
        <DarkCarouselWrapper>
          <Title intent="subdued">3</Title>
        </DarkCarouselWrapper>
      ),
    },
  ];
  const lightBgCollection = [
    {
      id: 1,
      content: (
        <LightCarouselWrapper>
          <Title>1</Title>
        </LightCarouselWrapper>
      ),
    },
    {
      id: 2,
      content: (
        <LightCarouselWrapper>
          <Title>2</Title>
        </LightCarouselWrapper>
      ),
    },
    {
      id: 3,
      content: (
        <LightCarouselWrapper>
          <Title>3</Title>
        </LightCarouselWrapper>
      ),
    },
  ];
  return (
    <Grid.Row gutter={['medium', 'none']}>
      <Grid.Col style={{ height: 300 }} span={[12, 12, 12, 12, 12]}>
        <Carousel
          collection={darkBgCollection}
          autoPlay
          selectedItemIndex={currentItemIndexDark}
          onItemIndexChange={setCurrentItemIndexDark}
        />
      </Grid.Col>
      <Grid.Col style={{ height: 300 }} span={[12, 12, 12, 12, 12]}>
        <Carousel
          collection={lightBgCollection}
          autoPlay
          variant="dark"
          interval={4000}
          selectedItemIndex={currentItemIndexLight}
          onItemIndexChange={setCurrentItemIndexLight}
        />
      </Grid.Col>
    </Grid.Row>
  );
}
```

### Infinite Carousel

Infinite carousel can be enabled by setting `infiniteLoop` to `true`. When it is enabled, the last slice will be connected to the first slice and vice-versa.

```jsx live

  const { Title } = Typography;
  const darkBgCollection = [
```
